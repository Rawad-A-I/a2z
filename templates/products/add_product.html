Add New Product

Step 1: Choose Product Type

Select the type of product you want to create

Standalone Product

A single product with fixed price

Example: iPhone, Laptop, Book
Parent Product

A product that will have size variants

Example: T-Shirt, Sneakers, Jeans
Size Variant

A specific size of an existing parent

Example: T-Shirt Large, Sneakers Size 10
Step 2: Product Details

Fill in the product information

Standalone Product: This product has a fixed price and no size variants.
Parent Product: This product will have size variants. You don't set a price here - prices are set for each size variant.
Base name for the product (e.g., "T-Shirt")
Description that applies to all size variants
Size Variant: This is a specific size of an existing parent product.
Examples: Small, Medium, Large, Size 10
